## 前端 - How To Install

1. 克隆本仓库

   ```bash
   git clone https://github.com/Xenorian/mirror_v2.git
   ```

2. 进入前端文件夹`mirrir_v2`

   ```bash
   cd mirrir_v2
   ```

3. 安装所需依赖

   ```bash
   npm install
   ```

4. 进入`src/stores`，修改`repoData.ts`，将后端ip与端口写入m_rooturl中

5. 开始运行

   ```bash
   npm run dev
   ```
